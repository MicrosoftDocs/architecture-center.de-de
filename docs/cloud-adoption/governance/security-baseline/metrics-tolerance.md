## <a name="risk-tolerance-indicators"></a><span data-ttu-id="4784e-134">Risikotoleranzindikatoren</span><span class="sxs-lookup"><span data-stu-id="4784e-134">Risk tolerance indicators</span></span>

- <span data-ttu-id="4784e-144">**Auslöser für geschäftskritische Workloads**.</span><span class="sxs-lookup"><span data-stu-id="4784e-144">**Mission-critical workloads trigger**.</span></span> <span data-ttu-id="4784e-145">Ein Unternehmen, das geschäftskritische Workloads in der Cloud bereitstellt, sollte in die Disziplin „Sicherheitsbaseline“ investieren, um mögliche Unterbrechungen des Diensts oder die Offenlegung sensibler Daten zu verhindern.</span><span class="sxs-lookup"><span data-stu-id="4784e-145">A company deploying mission-critical workloads to the cloud should invest in the Security Baseline discipline to prevent potential disruption of service or sensitive data exposure.</span></span>
- <span data-ttu-id="4784e-146">**Auslöser für geschützte Daten**.</span><span class="sxs-lookup"><span data-stu-id="4784e-146">**Protected data trigger**.</span></span> <span data-ttu-id="4784e-147">Ein Unternehmen, das Daten in der Cloud hostet, die als vertraulich, privat oder anderweitig rechtlich bedenklich eingestuft werden können.</span><span class="sxs-lookup"><span data-stu-id="4784e-147">A company hosting data on the cloud that can be classified as confidential, private, or otherwise subject to regulatory concerns.</span></span> <span data-ttu-id="4784e-148">Es ist eine Disziplin „Sicherheitsbaseline“ erforderlich, um sicherzustellen, dass diese Daten nicht verloren gehen, offengelegt oder gestohlen werden.</span><span class="sxs-lookup"><span data-stu-id="4784e-148">They need a Security Baseline discipline to ensure that this data is not subject to loss, exposure, or theft.</span></span>
